Simply clone the repo, and open main.html in any browser (maybe)

The buttons to control this are hidden at the bottom of the page, beautifully skinned, 2/3 of them work. Just use the edit function to remove nodes.

Edit Mode: click on nodes to move them, placing one on another merges them
Create Mode: Click on a node to begin an edge from it, or click anywhere to create an edge from a new node.

The initial, human friendly algorithm for extracting a knot from a graph can found at http://www.entrelacs.net/The-basics. Definetely worth a read. This program was created to give a general idea of what knot a graph would produce, before dedicating yourself to draw it.
